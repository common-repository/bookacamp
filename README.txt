Bookacamp for Wordpress - a plugin for easy integration of the booking system in Wordpress

== Description ==

Bookacamp is an installation-less booking system, developed especially for youth and kids travel. Bookacamp targets for tour and event operators, searching for a all-in-one solution to organize camps, manage accounting, manage camp counselors and everything else releated to your tour operators business. This plugin helps you to integrate the booking forms to your Wordpress installation without having any trouble with technical issues.

== Installation ==

From Wordpress Plugin directory

1. Search for bookacamp and install
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the necessary fields in the plugin settings menu

From file:

1. To install the plugin either upload the wp-bookacamp.zip file via the plugin installation interface
or extract the zip file and upload the folder `wp-bookacamp` to the `/wp-content/plugins` directory of your WordPress installation
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the necessary fields in the plugin settings menu

== Frequently Asked Questions ==

= Does the plugin provide shortcodes to display the result table? =

No. The Bookacamp element is available as an standalone element in the Wordpress site customizer when you edit the contents of your page/post.
